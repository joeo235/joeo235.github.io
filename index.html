Great things start small and take time to grow.

Hi, I'm Joe.

I'm a web content guy learning to become more of a web development guy. Currently I work at DePaul University in Chicago. Things I do a lot at work: writing web content, producing short videos, orchestrating email campaigns, managing a bunch of websites. As you might guess, a lot of my current projects stem from my work at DePaul. But I like to stay busy learning to code at Free Code Camp and hope to be adding more personal projects soon.

Projects

Tribute to Pierre Francois Olivier de Vezin

I created this webpage to honor my fifth-great paternal grandfather, who emigrated from France to Quebec, helping to establish what would be the first iron forge and industry town in Canada.

Teaching Commons

I took this website, originally created in an ugly (and now defunct) CMS called Serena Collage, and made it pretty, despite it being a SharePoint site. I also reworked the IA, written a bunch of content, and produced many of the videos too.

Office for Teaching, Learning and Assessment

Eek—this site was originally created and maintained in Dreamweaver (what a mess). I migrated the site over to SharePoint and revised the IA. Now visitors can see upcoming events, read the latest blog posts, and more.

Student Success

I created this site using DePaul's SharePoint framework (loosely based on Bootstrap with a few JS plugins for things like sliders, etc.). For the first time, students can view all the tutoring options in place, in addition to other kinds of support like mentoring, advising, and health services.

Learning in Progress

This is a blog for the Office for Teaching, Learning and Assessment that I set up using WordPress. I created a custom child theme and provided direction on the logo. I also occasionaly write blog posts—actually, I'm probably overdue for writing another one!

Teaching and Learning Certificate Program

This is another WordPress site that I got up and running for one of our office's certificate programs. I did a little editing of the WordPress loop in PHP and some custom styling. Check out the cool videos I helped produce.

Assessment Certificate Program

This is another WordPress site that is a part of our office's WordPress multisite installation, which I set up. And there are more cool videos to check out, produced by me and my student employee!
